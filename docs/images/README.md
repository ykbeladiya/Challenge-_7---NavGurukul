# Screenshots

This directory contains screenshots demonstrating various features of the Meeting-to-Modules system.

## Available Screenshots

### Theme List (`theme-list.png`)
Shows the output of `mtm list --themes` command displaying:
- Theme IDs
- Project names
- Top terms/keywords
- Support counts

### Generated Tutorial Module (`tutorial-module.png`)
Shows a generated tutorial module in Markdown format, including:
- Title and description
- Table of contents
- Step-by-step instructions
- Backlinks to source notes
- Version information

### Diff View (`diff-view.png`)
Shows the output of `mtm diff` command comparing two module versions:
- Unified diff format
- Context lines
- Added/removed sections
- HTML or Markdown diff output

### Exports Folder (`exports-folder.png`)
Shows the structure of the exports directory:
- Timestamped export folders
- CSV files (steps.csv, definitions.csv, faqs.csv, etc.)
- Markdown modules
- README.md with export information
- Optional PDF files

## Adding Screenshots

To add screenshots:

1. Run the pipeline to generate sample data
2. Capture screenshots of the various outputs
3. Save them in this directory with the names above
4. Ensure images are in PNG or JPG format
5. Update the README.md to reference these images
